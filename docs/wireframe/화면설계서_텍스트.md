### 자유게시판 목록
1. 상단 바 (Status Bar)
   기능 설명: 시스템 상태 정보 표시 (시간, 네트워크, 배터리)

인터랙션: 없음

상태: OS 제어 영역

이동 화면: 없음

2. 네비게이션 바 (Navigation Bar)
   기능 설명: '<' 뒤로가기, '자유게시판' + '인하대', 검색/더보기 아이콘

인터랙션: '<' 탭 → 이전 화면 | 검색 탭 → 검색 화면 | '⋮' 탭 → 설정 메뉴

상태: 게시판명/학교명 동적 표시

이동 화면: 이전 화면 / 검색 화면 / 게시판 설정

3. 광고 배너 (Ad Banner)
   기능 설명: 외부 광고 영역, 앱 아이콘/제목/설명 카드

인터랙션: 탭 → 외부 앱스토어 또는 광고 링크로 이동

상태: 자동 갱신, 로드 실패 시 숨김

이동 화면: 외부 링크 (앱스토어/웹)

4. 고정/추천 게시글 카드 (Pinned Cards)
   기능 설명: 가로 스크롤 카드 (질문글, 기숙사 등), 아이콘+제목+부제목

인터랙션: 탭 → 해당 게시글 상세 이동 | 좌우 스와이프

상태: 관리자 고정 게시글 표시, 0개 시 영역 숨김

이동 화면: 게시글 상세 (1.2)

5. HOT 인기 게시글 바 (Trending Bar)
   기능 설명: 분홍/빨간 배경, 🔥 + 인기 게시글 제목 + 공감수 배지(183)

인터랙션: 탭 → 해당 인기 게시글 상세 이동

상태: 실시간 자동 업데이트, 인기글 없을 시 숨김

이동 화면: 인기 게시글 상세 (1.2)

6. 게시글 피드 (Post Feed)
   기능 설명: 세로 스크롤 목록, 제목(굵게)+미리보기(회색)+메타정보

인터랙션: Pull-to-refresh | 무한 스크롤 페이지네이션

상태: 로딩/빈 목록/에러 상태 분기

이동 화면: 각 게시글 상세 (1.2)

7. 게시글 아이템 (Post Item)
   기능 설명: 제목+미리보기+메타(댓글수, 조회수, 시간, 익명), 썸네일 우측

인터랙션: 탭 → 게시글 상세 화면(1.2) 이동

상태: 썸네일 유/무, 새 글 배지 표시

이동 화면: 게시글 상세 (1.2)

8. FAB 글 쓰기 버튼 (Write FAB)
   기능 설명: 하단 중앙 부유 버튼 (빨간 배경, ✎ + '글 쓰기')

인터랙션: 탭 → 글 쓰기 화면(1.3) 모달 전환

상태: 스크롤 시 항상 고정, 비로그인 시 로그인 유도

이동 화면: 글 쓰기 (1.3)


### 게시글 상세
1. 상단 바 (Status Bar)
   기능: 시스템 상태 정보 표시 (시간 3:36, 네트워크 5G, 배터리 91%), OS 제어 영역

인터랙션: 없음. 사용자 조작 불가 영역

2. 네비게이션 바 (Nav Bar)
   구성: 좌측 '<' 뒤로가기, 중앙 '자유게시판', 우측 멀티링크(⋮) '더보기'

인터랙션: '<' 탭: 목록으로 이동. '⋮' 탭: 해당 게시글 알림 on/off 토글, 신고/삭제/공유 옵션 메뉴

이동 화면: '<' → 자유게시판 목록(1.1) / '⋮' → 신고/삭제/공유 액션시트

3. 작성자 정보 (Author Info)
   구성: 회색 아바타 + '익명' 닉네임(글쓴이 표시) + '03/27 15:22' 타임스탬프(회색)

인터랙션: 익명 게시판이므로 성별 비공개, 프로필 탭 불가

4. 게시글 본문 (Post Content)
   구성: 제목 "주 3시간 강의 4번 빠지면 결석 아님?" (18px, Bold) + 본문 "맞지?" (15px). 이미지 첨부 시 본문 하단 표시

인터랙션: 텍스트 선택/복사 가능, 이미지 탭 시 전체화면 보기

5. 액션 바 (Action Bar)
   구성: 공감, 댓글 3, 스크랩. 세 가지 버튼 가로 균등 배치

인터랙션: 공감 탭: +1/취소 토글. 댓글 탭: 댓글 영역으로 스크롤. 스크랩 탭: 내 스크랩 목록에 저장/해제

상태: 공감/스크랩 활성 시 아이콘 색상 변경(빨간색/노란색), 비활성: 회색

6. 광고 배너 (Ad Banner)
   기능: Spotify 학생 할인 광고. 녹색(#1DB954) 배경 + 텍스트 + 핸드폰 이미지

인터랙션: 탭: 외부 Spotify 페이지로 이동. 닫기(X) 버튼으로 광고 숨김 가능

7. 댓글 목록 (Comments)
   구성: 각 댓글: 아바타 + 닉네임(익명1/익명2) + 본문 + 타임스탬프. 우측 (⋮) 아이콘

인터랙션: '⋮' 탭: 댓글 공감, 답글 쓰기, 쪽지 보내기, 신고/삭제/공유 옵션. 세로 스크롤 가능

8. 대댓글 (Reply)
   구성: '↳' 아이콘으로 들여쓰기 표시. 부모 댓글 하단에 위치. 형식은 일반 댓글과 동일

인터랙션: 대댓글의 대댓글은 같은 레벨에 표시. 공감/신고 동일하게 작동

9. 하단 댓글 입력 바
   기능: 댓글 및 대댓글 작성을 위한 고정 하단 입력 바

구성: 텍스트 입력 필드 (placeholder: '댓글을 입력하세요.') + 익명 체크박스 + 전송 버튼

인터랙션: 입력 필드 탭 → 키보드 올라옴. 텍스트 입력 후 전송 버튼 탭 → 댓글 등록 및 목록에 추가. 익명 체크 시 '익명'으로 표시

상태: 비어있을 때 전송 버튼 비활성화(회색), 텍스트 입력 시 전송 버튼 활성화(빨간색)

이동 화면: 없음 (현재 화면에서 댓글 추가 후 목록 갱신)


### 글쓰기
1. 상태 바 (Status Bar)
   기능: 시스템 상태 정보 표시 (시간 3:37, 네트워크 5G, 배터리). OS 제어 영역.

인터랙션: 없음. 사용자 조작 불가 영역.

2. 네비게이션 바 (Nav Bar)
   구성: 좌측 '닫기' 버튼, 중앙 '글 쓰기' 제목(17px, Bold), 우측 '완료' 텍스트 버튼(파란색)

인터랙션: '닫기' 탭: 작성 취소 확인 다이얼로그 표시 → '작성을 취소하시겠습니까?' → 확인 시 목록(1.1)으로 이동. '완료' 탭: 입력 검증 후 게시글 등록 → 목록(1.1)으로 복귀.

상태: 제목/본문 모두 빈 상태: '완료' 버튼 비활성(회색). 제목만 입력: '완료' 활성(파란색). 등록 중: 로딩 스피너 표시.

이동 화면: '닫기' → 자유게시판 목록(1.1) | '완료' → 자유게시판 목록(1.1) + 새 게시글 상단 표시

3. 제목 입력 필드 (Title Input)
   기능: 게시글 제목 입력 영역. 플레이스홀더 '제목을 입력해주세요.' (20px, Bold). 하단 구분선으로 본문 영역과 분리.

인터랙션: 탭: 키보드 활성화 + 플레이스홀더 숨김 + 커서 표시. 한 줄 입력만 가능(줄바꿈 불가). 최대 50자 제한.

상태: 빈 상태: 플레이스홀더 표시. 입력 중: 실시간 텍스트 표시. 제목 미입력 시 '완료' 버튼 비활성.

4. 본문 입력 필드 (Body Input)
   기능: 게시글 본문 입력 영역. 플레이스홀더 '학교 친구들과 자유롭게 얘기해보세요. #수강신청 #취업' (15px). 멀티라인 입력 지원.

인터랙션: 탭: 키보드 활성화. 줄바꿈 가능. '#' 입력 시 해시태그 자동완성 표시. 세로 스크롤로 긴 본문 작성 가능. 본문 없이 제목만으로도 등록 가능.

5. 커뮤니티 이용규칙 안내 (Guidelines)
   구성: 상단: '커뮤니티 이용규칙 전체 보기 >' 링크 버튼 (회색 배경, 둥근 사각형). 하단: 규칙 요약 텍스트 (회색, 12px). 정치/사회 관련 행위 금지, 홍보/판매 관련 행위 금지 등 요약.

인터랙션: '전체 보기 >' 탭: 커뮤니티 이용규칙 전문 화면으로 이동 (웹뷰 또는 새 화면). 요약 텍스트: 세로 스크롤 가능. 첫 방문 시 자동 표시, 이후 접힘 가능.

목적: 사용자에게 게시판 규칙을 사전 안내하여 위반 게시글 작성을 예방. 위반 시 게시물 삭제 및 서비스 이용 제한 경고.

6. 규칙 요약 텍스트 (Rules Summary)
   내용: ■ 정치·사회 관련 행위 금지: 국가기관/정치단체/언론 언급, 정책·외교 의견, 성별·종교·인종 등 사회적 이슈. ■ 홍보 및 판매 관련 행위 금지: 영리/비영리 사업체 홍보, 바이럴 홍보. * 시사·이슈 게시판 전용 작성 가능.

표시 방식: 회색 텍스트 (12~13px), 선높이 1.5배. 글 쓰기 화면 중하단에 위치. 키보드 활성 시 스크롤로 접근 가능.

7. 하단 툴바 (Bottom Toolbar)
   구성: 좌측: 포토(사진) 아이콘 + 링크(클립) 아이콘. 우측: '질문' 체크박스(비활성) + '익명' 체크박스(활성, 빨간색). 상단 구분선으로 본문과 분리.

인터랙션: 포토/링크 탭: 카메라/갤러리 선택 액션시트 표시 → 이미지 첨부 (최대 10장). 링크 탭: URL 입력 팔레트 표시 → 링크 삽입. '질문' 탭: 질문글 태그 토글 (on/off). 활성 시 게시글에 '질문' 라벨 부여. '익명' 탭: 익명 여부 토글. 기본값 ON(빨간색 체크). OFF 시 실명 공개 경고 다이얼로그.

상태: 포토/링크: 회색 아이콘. 탭 시 액션시트 표시. 질문: OFF(회색) / ON(검정). 익명: ON(빨간색, 기본값) / OFF(회색). 키보드 활성 시에도 항상 화면 하단 고정.

비교: 익명 해제 시 확인 다이얼로그: '익명 해제 시 닉네임이 공개됩니다. 계속하시겠습니까?' → 확인/취소. 이미지 첨부 시 본문 영역 하단에 썸네일 미리보기 표시.
